# House Price Predictor (Flask + ML)

A web application built using Flask and Machine Learning
to predict house prices based on user inputs.

## Features
- Linear Regression model
- Flask backend
- HTML/CSS frontend
- End-to-end ML deployment

## Run Locally
pip install -r requirements.txt  
python train_model.py  
python app.py
